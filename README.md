## ✨ Overview

**Fluency Hub** is a state-of-the-art landing page and course repository designed specifically for Vietnamese learners aiming for English mastery. It serves as the central bridge between our beautifully crafted educational content and the **Fluency Mobile App**. 

With a focus on **visual excellence** and **user experience**, Fluency Hub provides a seamless way to discover, filter, and download high-quality English courses tailored for various proficiency levels and specific skills.

## 🚀 Key Features

- **🎯 Dynamic Skill Discovery**: An intelligent tab bar that automatically categories courses based on actual content data.
- **📱 Responsive by Design**: Fully optimized for every screen size, from high-resolution desktops to mobile devices.
- **✨ Premium UI/UX**: Built with modern aesthetics, including glassmorphism effects, smooth transitions, and vibrant themed palettes.
- **📊 Real-time Stats**: Track course popularity and skill trends within the ecosystem.
- **🔒 Privacy First**: Integrated privacy and terms pages that reflect our commitment to local-first data principles.
- **🛠️ Course Management**: Powerful service layer for discovering and fetching courses from multiple decentralized sources.

## ⚙️ Getting Started

### Prerequisites

- **Node.js**: `20.x` or higher
- **npm**: `10.x` or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/KhanhRomVN/Fluency.git
   cd Fluency/FluencyHub
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
